# [Tipsy Admin](http://www.tipsy.co.kr/admin)

## 개요 
Tipsy 서비스에서 사용되는 데이터 관리 페이지 입니다. <br><br>

## 기능
> 대시보드

- Daily Goal
- 신규 데이터 타임라인
- 관리 로그

> 데이터 조회/입력/수정/삭제

- 주류
- 칵테일
- 재료
- 장비
- 용어

<br />
